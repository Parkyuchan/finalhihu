# 🎓학교 생활 서비스

***서비스 기간***: 2022. 11.15 ~ 2022.12.23  
***일 평균 방문자 수***: 약 50명  
***Framework***: <img src="https://img.shields.io/badge/Django-092E20?style=flat-square&logo=django&logoColor=white"/>  
***Language***: <img src="https://img.shields.io/badge/Python-3776AB?style=flat-square&logo=python&logoColor=white"/> <img src="https://img.shields.io/badge/HTML5-E34F26?style=flat-square&logo=html5&logoColor=white"/> <img src="https://img.shields.io/badge/CSS3-1572B6?style=flat-square&logo=css3&logoColor=white"/>  

## 📌 서비스 개요:
**교내·외에 흩어져 있던 정보를 하나의 웹 서비스로 통합하여
학생들이 학교 생활에 필요한 정보를 빠르게 확인할 수 있도록 만든 서비스입니다.**

**제공 기능**
1. 교내 사무실 정보 및 전화번호
2. 학부 및 학과 정보
3. 학부·학과·학교 부서별 연락처
4. 동아리 정보
5. 학사 일정 및 공지
6. 운영자 공지
7. 교내 식당 메뉴 정보
8. 통학 버스 예매 페이지 연동
9. 주변 식당 정보
10. 지역 시내/외 버스 시간 정보
11. 문의 메일 전송
12. 학교 홈페이지 바로가기
13. 학생 간 소통 공간 제
